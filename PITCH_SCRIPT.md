# Pitch Script — Competency Intelligence Demo

A 5-minute walkthrough script, screen by screen, for live demo to a VP
audience (Sber default; swap the bracketed lines for VK/Rosatom/Yandex).
Each section: **[click]** = what you do, *"..."* = what you say.

Total runtime target: 5 minutes. Practice it once against a timer before
the real room — it's easy to linger on Screen 6 because the animation is
fun to watch.

---

*"I'm not going to show you an HR system. I'm going to show you how we
answer one question for the board: do we have the people to execute our
AI strategy — and if not, what do we do about it in the next 90 days."*

## If you only have 90 seconds (elevator version)

Dashboard → click "Проанализировать организацию" → skip straight to
Screen 6 (Orchestration), let it auto-play while you say the "six agents,
not one black box" line → Screen 8 (Division Readiness) → close.

## If a VP interrupts to ask "is this real AI or a mockup?"

Be straight about it: Screen 2 (Employee Analysis) and the equivalent
buttons in VK/Yandex mode call Claude live when an API key is configured;
the rest of the data is realistic mock data standing in for what would be
pulled from HR systems, Confluence, and project tools in a real pilot. The
orchestration *pattern* on Screen 6 is real architecture, not staged —
that's genuinely how a multi-agent pipeline like this would be built.

## Known soft spots — rehearse around these

- Screen 6's auto-play takes about 8-9 seconds total. If you start talking
  immediately, you'll finish your line before agent 3 fires. Let the first
  beat breathe, then talk.
- If you're demoing on a conference room screen smaller than your laptop,
  test the mobile/narrow layout beforehand — grids reflow but you should
  know where things land before you're live.
- Don't open Screen 3 (Mentor Matching) cold without Screen 2 first — the
  "based on these specific gaps" line only lands if the audience just saw
  the gaps.
